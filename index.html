При первом запуске git ,может запросить устоновить конфигурацию.В виде электроного адреса имейла,а также имя пользователя.

  • git config --global user.email "user@email.com" Устоновит электроную почту пользователя.
  • git config --global user.name "user Name" Устоновит имя пользователя.

Основные команды git.

  • git clone [path/file] Клонировать репазиторий
  • git status показывает статус текушего репазитория
  • git add [path/file] Добавить новый файл под контроль git
  • git commit -am "coment" Закрепить изменения в файле
  • git push отправить файлы на сервер
  • git pull получить с сервера файлы
  • git init Создаст пустой репозиторий в папке
  • cat [path/file] Выводит содержания файла
