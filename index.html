При первом запуске git ,может запросить Установить конфигурацию.В виде электроного адреса имейла,а также имя пользователя.

  • git config --global user.email "user@email.com" Установит электроную почту пользователя.
  • git config --global user.name "user Name" Установит имя пользователя.
  • git config --global core.editor [path/exe file] Установит текстовый редактор или изменит его.

Основные команды git.

  • git clone [path/file] Клонировать репазиторий
  • git status показывает статус текушего репазитория
  • git add [path/file] Добавить новый файл под контроль git
  • git commit -am "coment" Закрепить изменения в файле
  • git push отправить файлы на сервер
  • git pull получить с сервера файлы
  • git init Создаст пустой репозиторий в папке
  • cat [path/file] Выводит содержания файла
  • git help Выводит список команд
  • git show [0ght6s] Выводит текуший комит
  • git add [.] Добавит все деректории и файлы под контроль git
  • git reset HEAD [path/file] Позволяет исключить из деректории git
  • Создания файла .gitignor, можно в писать туда файлы и деректории которые не будут обрабатыватся git
  • git log Просмотр ветки коммитов
